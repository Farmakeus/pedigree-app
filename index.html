    📚 Yardım & Klavye Kısayolları

    ⌨️ Klavye Kısayolları

    • Ctrl + Z — Geri Al
    • Ctrl + Y / Ctrl+Shift+Z — İleri Al
    • Ctrl + S — Pedigri'yi kaydet (JSON)
    • Ctrl + A — Tüm bireyleri seç
    • Tab — Sonraki bireye geç (klavye gezinti)
    • Enter — Odaklı bireyi düzenle
    • Delete — Odaklı bireyi sil
    • ? — Bu yardım panelini aç
    • Esc — Aktif aracı/paneli kapat

    🖱️ Fare İşlemleri

    • Şekle tıkla — Düzenleme paneline geç
    • Şeklin üzerine gel — Bilgileri tooltip'te gör
    • Shift + Tıkla — Çoklu seçim (toplu işlem için)
    • Bağla aracı — Şekiller arası çizgi çek (aynı nesil = eş, farklı nesil = ebeveyn-çocuk, yeşil nokta = kardeş ekle)
    • Taşı aracı — Şekli sürükle, Y otomatik nesil hizasına oturur
    • Silgi aracı — Şekle tıkla → silinir (Geri Al ile dönülebilir)

    🧬 Sembol Anlamları

    • □ Erkek    ○ Kadın    ◇ Bilinmiyor
    • ■ / ● Etkilenmiş (Hasta)
    • ⊡ / ⊙ Taşıyıcı
    • Çapraz çizgi = Vefat
    • Tek çizgi (—) Eş · Çift çizgi (═) Akraba evliliği · Çift slash (//) Boşanmış
    • Köşeli parantez [□] Evlat edinme
    • ♢P Devam eden gebelik · ▼ SAB (düşük) · ▼/ TOP (tahliye) · SB Ölü doğum

    💾 Veri Yönetimi

    • Otomatik kayıt: Tüm değişiklikler tarayıcıda otomatik saklanır
    • Kaydet: JSON dosyası olarak indir (yedekleme/paylaşım)
    • Yükle: Daha önce kaydettiğin JSON'u geri yükle
    • Paylaş: Pedigri'yi URL içinde kodlayıp link olarak paylaş
    • Export: SVG / PNG / PDF / CSV olarak çıktı al

    🧬 Modlar

    • 🧬 Nadir Hastalık: Mendel kalıtım, tek hastalık/fenotip alanı
    • 🩸 Kanser Pedigresi: Onkogenetik, çoklu kanser (max 3) + tanı yaşı + bilateral
